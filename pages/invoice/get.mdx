# Get

The Invoice Get method retrieves an array of invoices (one for each month of repayment) of a given Plan.

```bash filename="Endpoints" copy
GET /api/invoice/:pid
```

```ts filename="Response Object" {/* {1-3} */} copy
type Response = {
  id: number
  pid: number
  due: string
  amnt_due: number
  total: number
  fulfilled: boolean
}[]
```
