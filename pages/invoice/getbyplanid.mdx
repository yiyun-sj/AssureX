# GetByPlanId

The Invoice GetByPlanId method retrieves an array of invoices (one for each month of repayment) of a given Plan.

```ts filename="Request Object" {/* {1-3} */} copy
type Request = {
  id: string
}
```

```ts filename="Response Object" {/* {1-3} */} copy
type Response = {
  id: number
  pid: number
  due: string
  amnt_due: number
  total: number
  fulfilled: boolean
}[]
```
