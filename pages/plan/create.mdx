# Create

After a client chooses an option from the request method, they can call the Create method to confirm and begin that Plan.

```ts filename="Request Object" {/* {1-3} */} copy
type Request = {
  email: string
  date: string
  principal: number
  base: number
  interest: number
  installments: number
}
```

```ts filename="Response Object" {/* {1-3} */} copy
type Response = {
  id: string
  email: string
  date: string
  principal: number
  base: number
  interest: number
}
```
