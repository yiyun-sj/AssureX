# GetByEmail

The Plan GetByEmail method retrieves an array of plans given an email address used to identify the Plan owner.

```ts filename="Request Object" {/* {1-3} */} copy
type Request = {
  email: string
}
```

```ts filename="Response Object" {/* {1-3} */} copy
type Response = {
  id: number
  email: string
  date: string
  principal: number
  base: number
  interest: number
}[]
```
