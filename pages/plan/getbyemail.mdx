# GetByEmail

The Plan GetByEmail method retrieves an array of plans given an email address used to identify the Plan owner.

```bash filename="Endpoints" copy
GET /api/plan/email/:email
```

```ts filename="Response Object" {/* {1-3} */} copy
type Response = {
  id: number
  email: string
  date: string
  principal: number
  base: number
  interest: number
}[]
```
